Política de Privacidade e Proteção de Dados

1. Coleta de Dados

Coletamos dados necessários para o cadastro (nome, OAB, e-mail) e dados operacionais inseridos pelo usuário (processos e clientes)[cite: 53, 634].

2. LawTech Pro como Operadora

Em relação aos dados de terceiros inseridos no sistema, a LawTech Pro atua na qualidade de OPERADORA, nos termos da Lei 13.709/2018 (LGPD).

3. Armazenamento Seguro

Os dados são armazenados em infraestrutura de nuvem de alta segurança (Google Cloud ou AWS), com acesso restrito e criptografado[cite: 669].

4. Foro Elegido

Para dirimir quaisquer controvérsias, fica eleito o foro da Comarca de Feira de Santana, Bahia[cite: 778].
